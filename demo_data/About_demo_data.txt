Demo data for BorderStrength
============================

MICRO method (diagonal-only, fine-scale) -- synthetic chr II, 250 bp,
three engineered boundaries:
  demo_map.txt.gz       text map format (loc1  loc2  score, loc=chr:start:end)
  demo_hic200.txt.gz    hic200-cpp contact format (bin1  bin2  score)
  demo_bindef.txt       bin definition for the hic200-cpp format
  expected_demo_BS.txt        expected score output
  expected_demo_domains.txt   expected domain intervals

LARGE method (full matrix, cohesin/condensin/TAD) -- synthetic chr II,
10 kb dense matrix, two engineered boundaries:
  demo_large.matrix           dense contact matrix (rownames/colnames chr:start:end)
  expected_large_BS.txt       expected score output
  expected_large_domains.txt  expected domain intervals

Reproduce everything:
  sh demo_data/run_demo.sh

Or individually:
  sh BS.sh --method micro -i demo_data/demo_map.txt.gz    -o micro_BS.txt --window 3000
  sh BS.sh --method large -i demo_data/demo_large.matrix   -o large_BS.txt --window 100kb
  Rscript Draw_BS.R -i large_BS.txt -o large.png --chr II --domain large_BS_domains.txt

Small differences in the last shown digit can occur across BLAS/locale settings.
