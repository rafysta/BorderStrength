Demo data for BorderStrength
============================

The demo is a small synthetic single-chromosome (chr II) contact map at 250 bp
resolution with three engineered domain boundaries (at bins 140, 300 and 455).
It is provided in the two input formats BorderStrength accepts:

  demo_map.txt.gz      text map/matrix format (loc1  loc2  score, loc=chr:start:end)
  demo_hic200.txt.gz   hic200-cpp contact format (bin1  bin2  score)
  demo_bindef.txt      bin definition for the hic200-cpp format (bin  chr  start  end)

  expected_demo_BS.txt expected output (chr start end BS BS.norm boundary)

Reproduce the expected output:

  # map / matrix format
  sh BS.sh -i demo_data/demo_map.txt.gz -o demo_BS.txt --window 3000

  # hic200-cpp format (identical result)
  sh BS.sh -i demo_data/demo_hic200.txt.gz --bin demo_data/demo_bindef.txt \
           -o demo_BS.txt --window 3000

Both commands run in a few seconds and reproduce expected_demo_BS.txt
(small differences in the last shown digit can occur across BLAS/locale settings).
